# Burguer Thunder

## Descrição

Este é um exemplo de aplicativo de pedidos de hambúrgueres, desenvolvido em React.js, que permite aos usuários fazerem pedidos e visualizarem os pedidos feitos anteriormente.

## Funcionalidades

- **Fazer Pedidos**: Os usuários podem fazer pedidos, inserindo seu nome e o pedido desejado.
- **Visualizar Pedidos Anteriores**: Os usuários podem visualizar os pedidos feitos anteriormente.
- **Excluir Pedidos**: Os usuários podem excluir pedidos feitos anteriormente.

## Tecnologias Utilizadas

- React.js
- styled-components
- axios
- react-router-dom


# Como Usar

- Na página inicial, faça um novo pedido inserindo o nome do cliente e o pedido desejado.
- Na página de pedidos feitos, você pode visualizar todos os pedidos anteriores e excluir pedidos específicos clicando no ícone de lixeira.
